选中 Project Assets 中图片，查看属性 Pivot ,
点击按钮 Sprite Editor ,
选择 Pivot 中的 Custom 
调整自定义中心点 Custom Pivot x,y
应用设置点击 apply
最后将 Sprite Renderer 下的 Sprite Sort Point 设置为 Pivot
